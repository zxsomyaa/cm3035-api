CM3035 Coursework – Death Rates from Air Pollution API

Title URL What it does
Get all death rates /api/deathrates/ Returns all death rate records.
Filter by country /api/deathrates/country/<code>/ Returns records for a specific country.
Filter by year /api/deathrates/year/<year>/ Returns records for a specific year.
Country + year /api/deathrates/country/<code>/year/<year>/ Returns records for a given country and year.
Global average /api/deathrates/global-average/<year>/ Returns global average death rate.
Create record /api/deathrates/ Creates a new record (POST).
